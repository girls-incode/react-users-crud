### Features:

* Log in
* Sign up
* Create a User Table List
* Allow paginating on User Table List
* Search user by name
* Add a new user
* Edit a user
* Delete a user

#### To run:

#### `yarn start`

#### `yarn test`

#### `yarn build`

### Tech stack:
* react
* react-redux
* redux-thunk
* material-ui
* sass
* axios
